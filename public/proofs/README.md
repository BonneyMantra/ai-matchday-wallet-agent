# Proof Artifacts

Scripts write proof JSON here for the Vite demo to expose as static artifacts.

- `agent-decision.json`: generated by `npm run agent:pick`
- `xlayer-rpc-smoke.json`: generated by `npm run rpc:smoke`
- `local-anvil-proof.json`: generated by `npm run anvil:proof`
- `xlayer-deployment.json`: generated only when a funded X Layer deployer is supplied
- `xlayer-agent-submit.json`: generated only when a funded agent key and contract are supplied
