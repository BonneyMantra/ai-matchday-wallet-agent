# Build Plan

## Scope

Build a real, testable MVP for AI Matchday Wallet Agent:

1. Solidity contract for agent predictions and settlement.
2. Local/anvil proof showing wallet-signed prediction and settlement.
3. X Layer testnet RPC smoke proof.
4. Optional X Layer deploy/submit scripts gated by funded signer env vars.
5. Vite React UI showing the primary demo flow and proof artifacts.
6. README, execution packet, repo/submission plans, truth audit, and quality gate.

## Product Claim

The app is a World Cup-themed software agent wallet that submits non-cash match predictions and maintains an on-chain accuracy record. Until a funded X Layer transaction exists, the live X Layer claim is blocked and the working proof is local/anvil.

## Stack

- Foundry for Solidity tests and local chain proof.
- Vite + React + TypeScript for UI.
- `viem` for scripts and RPC calls.
- Static proof JSON under `proofs/`, generated by scripts.
